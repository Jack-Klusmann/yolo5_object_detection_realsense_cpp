# yolov5 Inference C++

Usage:
```
git clone https://github.com/JustasBart/yolov8_CPP_Inference_OpenCV_ONNX
cd yolov8_CPP_Inference_OpenCV_ONNX
```

```
mkdir build
cd build
cmake ..
make
./Yolov8CPPInference
```
